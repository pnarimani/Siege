## Doc Selection Guide

1. Need prefab structure and base components: read package index and basic layout.
2. Need auto-resize behavior: read auto-layout overview, layout element, content size fitter.
3. Need responsive scaling: read multiple resolutions and canvas scaler docs.
4. Need runtime prefab instantiation: read create-from-scripting.
5. Need world-space menu or diegetic panel: read world-space UI guide.
